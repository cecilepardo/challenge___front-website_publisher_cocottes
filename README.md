# challenge___front-website_publisher_cocottes

This project is a short frontend challenge completed in two days, using only HTML and CSS.

## Purpose

The goal of this project was to practice building a complete static website in a limited timeframe, focusing on structure, layout, and visual consistency without relying on JavaScript.

## What I practiced

- Structuring semantic HTML pages
- Building multi-page navigation (`index`, `albums`, `about`, `contact`)
- Creating layouts using CSS
- Managing spacing, typography, and visual hierarchy
